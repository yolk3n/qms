Формулярный список номенклатуры

Документ Формулярный список номенклатуры служит для отражения списка лекарственных средств и изделий медицинского назначения, которые используются организацией для оказания медицинских услуг. Документ Формулярный список номенклатуры позволяет ограничивать список номенклатуры при формировании документа Требование отделения.

Использование формулярных списков в программе управляется опцией Формулярные списки номенклатуры в настройках раздела Администрирование - Организации.

Табличная часть документа содержит список аптечных товаров с указанием для каждого единицы измерения и значения доступа. Значение доступа определяет доступность строки формуляра организации для конкретных отделений. Для изменения перечня отделений, которым доступна строка формуляра, необходимо нажать кнопку Доступ на закладке Товары. В появившейся форме следует выбрать значение реквизита Значение доступа (Доступно всем или Запрещено всем) и настроить исключения для отдельных отделений.

При заполнении табличной части возможны 3 варианта подбора:

  • по действующему веществу (МНН): В этом случае степень свободы при последующем подборе номенклатуры в заказе отделения будет максимальной (если при этом не заполнены реквизиты "Торговое наименование" и "Номенклатура"). Также реквизит "Действующее вещество МНН" будет являться фильтром для выбора реквизитов "Торговое наименование" и "Номенклатура".
  • по торговому наименованию: В этом случае при последующем подборе номенклатуры в заказе отделения в общем случае может быть подобрана любая подходящая номенклатура (если при этом не заполнен реквизиты "Номенклатура"), имеющая соответствующее торговое наименование (упаковки и производители могут быть произвольными). Также реквизит "Торговое наименование" будет являться фильтром для выбора реквизитов "Номенклатура".
  • по номенклатуре: В этом случае при последующем подборе номенклатуры в заказе отделения отбор будет установлен по конкретной номенклатуре.

При проведении документа информация о формулярном списке аптечных товаров организации записывается в регистр Периоды действий формуляра.

Для документа предусмотрен бизнес-процесс согласования и утверждения, который запускается по кнопке на панели инструментов документа, если в настройках системы в разделе Администрирование - Организации установлен флажок Согласование и утверждение формуляров. При этом статус документа автоматически меняется. Реквизит Статус может принимать значения:

  • не согласован - устанавливается при создании нового документа:
  • на согласовании - запущен процесс согласования, документ будет иметь этот статус, пока не пройдет согласование со всеми исполнителями бизнес-процесса согласования;
  • согласован - прошел процесс согласования со всеми исполнителями;
  • на утверждении - запущен процесс утверждения;
  • не утвержден - формулярный список не утвержден;
  • утвержден - формулярный список утвержден всеми исполнителями бизнес-процесса утверждения. Все утвержденные формулярные списки за конкретный промежуток времени можно увидеть по ссылке Формулярный список в разделе Управление аптекой.

Статус документа изменяется автоматически при прохождении бизнес-процесса. Пользователь с ролью Ручное изменение статуса формулярных списков номенклатуры имеет право изменить статус документа вручную. Также вручную меняется статус документа, если для согласования и утверждения формулярных списков не используются бизнес-процессы.
