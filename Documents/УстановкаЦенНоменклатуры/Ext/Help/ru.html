Установка цен номенклатуры

Документ Установка цен номенклатуры предназначен для фиксирования цен для товарных позиций.
В шапке документа надо указать вид цены, формируемый данным документом.
Документом Установка цен номенклатуры , введенным на основании документа Поступление товаров, можно зафиксировать закупочные цены, которые могут являться базовыми для расчета розничных цен.
Чтобы установить новые значения цен, надо выбрать вид устанавливаемой цены, заполнить табличную часть документа, затем по кнопке "Изменить цены" указать способ, которым надо рассчитать новые цены выбранных позиций номенклатуры.

Табличную часть документа можно заполнить подбором товаров, либо в соответствии с установленными параметрами отбора по кнопке Изменить строки - Добавить товары по отбору и перенести информацию об этих товарах в табличную часть, либо заполнить товарными позициями из заранее подготовленного файла.

Для изменения цен, которые уже указаны в документе, можно использовать для выделенных строк табличной части команду Изменить цены - Изменить на процент. При выполнении этой команды можно изменять все цены, в том числе и те, которые были рассчитаны по формулам.
