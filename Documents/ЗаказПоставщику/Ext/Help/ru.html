Заказ поставщику

Документ Заказ поставщику предназначен для фиксации предварительной договоренности о приобретении у поставщика позиций номенклатуры и может являться документом, на основании которого производится получение позиций номенклатуры.

Подбор товаров на закладке Товары можно произвести из справочника Номенклатура по кнопке . или Заполнить-Подобрать товары .

В табличной части документа на закладке Товары можно задать дату ожидаемого поступления каждой номенклатурной позиции. если снять флажок флажок Поступление одной датой под табличной частью документа. Если ожидается. что товары по данному заказу поступят одновременно, то флажок должен быть установлен.

В программе существует возможность формирования заказов поставщикам на основании документа Спецификация к договору. В таком случае подбор товаров при формировании заказа поставщику осуществляется с учетом перечня и количества позиций в спецификации к договору. Если заказ выполнен не полностью, то пользователь должен указать причину, по которой невозможно было обеспечить заказ полностью (отменено по причине).

При проведении документа по регистру накопления формируются соответствующие записи, отражающие в программе факт заказа товаров у поставщика. При проведении документа контролируется корректность заказанного количества товара с количеством в документе-основании Спецификация к договору.

Для детального отслеживания статусов заказов поставщикам в разделе Администрирование - Закупки надо установить флажок Статусы заказов поставщикам, тогда в шапке документа появляется реквизит Статус. Статус устанавливается вручную и может принимать значения:

  • Не согласован - устанавливается для нового документа;
  • Согласован - заказ согласован;
  • Подтвержден - заказ подтвержден поставщиком;
  • К поступлению - на основании документа Заказ поставщику с этим статусом можно вводить документ Поступление товаров. Количество товаров в документе будет отражаться в отчетах Состояние выполнения спецификации и Состояние выполнения заказа в графе К поступлению.
  • Закрыт - по данному заказу поступили все товары.

В реквизите Приоритет можно указать приоритет данного документа. По кнопке Еще - Изменить форму можно вывести этот реквизит в форму списка документов, а по кнопке Еще - Настроить список можно задать условное оформление для заказов высокого приоритета.

В документе можно анализировать состояния выполнения заказа с помощью отчета Состояния выполнения заказов поставщикам.
Отчет формируется как из списка так и из формы документа по кнопке .
Из спика документов отчет можно формировать сразу по нескольким выделенным документам.
