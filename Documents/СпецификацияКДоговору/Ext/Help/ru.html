Спецификация к договору

Документ Спецификация к договору предназначен для фиксации спецификации (приложения) к договору на поставку поставщиком позиций номенклатуры аптечных товаров.

Табличную часть документа можно заполнить по кнопке Добавить или Подобрать товары , при этом для выбора товаров откроется справочник Номенклатура.

На основании спецификации к договору можно формировать заказы поставщику. В таком случае подбор позиций при формировании заказа поставщику осуществляется с учетом перечня и количества позиций в спецификации к договору.

Для детального отслеживания статусов спецификаций к договорам в разделе Администрирование - Закупки надо установить флажок Статусы спецификаций к договорам, тогда в шапке документа появляется реквизит Статус. Статус устанавливается вручную и может принимать значения:

  • Не согласован - для нового документа
  • Согласован - спецификация прошла процедуру согласования
  • К выполнению - этот статус устанавливается. когда по данной спецификации можно подготовить заказ поставщику, т.е. документ Заказ поставщику можно ввести на основании документа Спецификация к договору только со статусом К выполнению.
  • Закрыт - по данному документу получены все товары.

С помощью отчета Состояние выполнения спецификаций к договорам можно получить информацию о динамике исполнения государственного контракта по каждой позиции номенклатуры с учетом заказанных, оформленных к поступлению в документе Заказ поставщику и поступивших от поставщика номенклатурных позиций.

Отчет формируется по кнопке Отчеты как из формы документа, так и на командной панели формы списка документов. Во втором случае отчет можно сформировать по нескольким документам, предварительно их выделив.
