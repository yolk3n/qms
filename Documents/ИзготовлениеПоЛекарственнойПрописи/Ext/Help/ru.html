Изготовление по лекарственной прописи

Документ Изготовление по лекарственной прописи служит для отражения в системе факта внутриаптечного изготовления лекарственных средств по лекарственным прописям.
В шапке документа в группе Продукция указывается данные выпускаемого товара: номенклатура продукции, серия (если по номенклатуре ведется учет по сериям), количество выходной продукции, единица измерения выходной продукции и лекарственная пропись из справочника "Лекарственные прописи", по которой будет проводиться изготовление.

Закладка Пропись

На закладке Пропись указывается перечень ингредиентов, которые требуются для изготовления по лекарственной прописи. Перечень ингредиентов можно заполнить автоматически на основании содержимого элемента справочника Лекарственные прописи по кнопке Заполнить пропись. С помощью кнопки Добавить можно дополнить состав ингредиентов прописи.

Закладка Товары

Для подбора номенклатурных позиций по остаткам на складе аптеки в соответствии с закладкой Пропись следует нажать кнопку Подобрать товары. При этом будет выведена вспомогательная форма подбора по прописи, которая позволяет подбирать номенклатуру как в ручном, так и автоматическом режиме, форма позволяет переносить данные на закладку Товары документа Изготовление по лекарственной прописи.

При проведении документа Изготовление по лекарственной прописи данные отражаются в регистрах накопления по учету количества товаров на складах, в регистрах по выпуску продукции. Стоимость списанных компонентов и изготовленного товара рассчитывается обработкой Расчет себестоимости. При проведении документа контролируется наличие в свободном остатке товаров для изготовления в том количестве, которое указано в документе.

Для документа Изготовление по лекарственной прописи предусмотрена печатная форма Акт выпуска продукции.
