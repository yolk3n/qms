Списание по назначениям врачей

Документ Списание по назначениям врачей предназначен для оформления списания товаров из отделения по выполненным назначениям врачей.

На закладке Товары располагаются два раздела:
  • Исполненные назначения: табличная часть заполняется по кнопке Заполнить формальными описаниями назначенных препаратов из документов Назначения врача. В табличную часть попадают только те товары, по которым в назначениях есть отметка о выполнении.
  • Товары к списанию: в табличную часть по кнопке Подобрать товары подбирается номенклатура, которая была фактически списана по назначению врача.

Документ Списание по назначениям врачей оформляется отдельно по каждому складу и отделению медицинской организации. В документе указываются статья и аналитика расходов, на которые должны относиться расходы по списанию товаров.

При проведении документа контролируется наличие в свободном остатке товаров в том количестве, которое указано в документе. Себестоимость списываемой номенклатуры рассчитывается обработкой Расчет себестоимости.
