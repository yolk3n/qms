Распределение расходов на поступления

Форма предназначена для распределения суммы дополнительных расходов на себестоимость товаров в выбранных документах поступления.
В форме указывается сумма к распределению на строки документов поступления.

Доступны варианты распределения по количеству и сумме.
Распределение производится на строки документов с типом номенклатуры Товар.

По гиперссылке Расшифровать документы по номенклатуре формируется отчет Расшифровка поступлений по номенклатуре предназначенный для анализа строк, на которые будет осуществлено распределение сумм дополнительных расходов,
