Чек ККМ

Документ предназначен для оформления продаж товаров в розницу. Продажа товаров может производиться только с помощью кассы ККМ, в которой установлен тип кассы Фискальный регистратор и указан склад типа Розничный магазин. Расчет с покупателем производится наличными денежными средствами или платежной картой.

Доступ к рабочему месту кассира может осуществляться с использованием персональных карт сотрудников со штрихкодом/магнитным кодом (при условии, что для данного рабочего места кассира в разделе Администрирование - Продажи - Розничные продажи включена возможность Использовать авторизацию).
В этом случае работа с рабочим местом кассира будет возможна только при наличии у сотрудника персональной карты.

Предварительные действия по оформлению чеков ККМ
  • Указать кассу ККМ в списке.
  • Открыть кассовую смену.
  • Зарегистрировать внесение денежных средств в кассу ККМ, если это необходимо.

Оформление чеков ККМ


Товары в табличную часть документа могут быть добавлены с использованием сканера штрихкодов или с использованием механизма ручного подбора товаров в документ.

При подборе товара в табличную часть документа для товаров автоматически устанавливаются отпускные (розничные) цены. В качестве розничных цен склада используются цены, зарегистрированные для того вида цен, который указан в карточке того склада, с которого оформляется чек ККМ.

Перед оформлением оплаты по чеку предусмотрена возможность отложить оформленный чек.

Для увеличения скорости обслуживания клиентов предусмотрено использование горячих клавиш.

Корректировка в чеке ККМ


Для случаев когда в документе при оформлении продажи по каким-либо причинам требуется удалить строки, предусмотрен механизм авторизации (требуется предварительное включение в справочнике Настройка РМК), предоставляющей возможность по быстрой смене прав кассира без необходимости перезапуска системы.

Закрытие смены


Документы вида Чек ККМ оформляются в течение кассовой смены. По окончании рабочего дня производится закрытие кассовой смены. При этом все документы Чек ККМ оформленные за день не удаляются, а остаются в информационной базе. Если в настройках параметров учета установлен флаг Архивировать чеки ККМ при закрытии кассовой смены, то документы архивируются.
При закрытии кассовой смены формируется и проводится обобщающий документ Отчет о розничных продажах, который подводит итоги розничной торговле по конкретной кассе ККМ.

Оформление возвратов


Возврат товаров розничному покупателю в течение кассовой смены оформляется документом Чек ККМ на возврат и может быть оформлен непосредственно кассиром по команде Оформить возврат. Если кассовая смена уже закрыта, то возврат товаров розничному покупателю оформляется с помощью документа Возврат товаров от клиента с установленным видом операции Возврат от розничного покупателя.

Работа с подключаемым оборудованием


Торговое оборудование, такое как сканеры штрихкодов, фискальные регистраторы, электронные весы и.п. должно быть предварительно подключено Администратором информационной базы в рабочем месте Подключаемое оборудование.
