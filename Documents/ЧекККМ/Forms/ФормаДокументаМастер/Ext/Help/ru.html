Рабочее место кассира

Рабочее место кассира предназначено для оформления продаж товаров в розницу. Продажа товаров может производиться только с помощью кассы ККМ, в которой установлен тип кассы Фискальный регистратор и указан склад типа Розничный магазин. Расчет с покупателем производится наличными денежными средствами или платежной картой.

Доступ к рабочему месту кассира может осуществляться с использованием персональных карт сотрудников со штрихкодом/магнитным кодом (при условии, что для данного рабочего места кассира в справочнике  Настройка РМК в разделе Администрирование - Продажи - Розничные продажи включена возможность Использовать авторизацию).
В этом случае работа с рабочим местом кассира будет возможна только при наличии у сотрудника персональной карты.

Перед началом ввода чеков ККМ в списке документов Чеки ККМ необходимо произвести следующие действия:

  • Указать кассу ККМ в списке документов Чеки ККМ.
  • Открыть кассовую смену.
  • Если это необходимо, то зарегистрировать внесение денежных средств в кассу ККМ (при этом автоматически формируется документ Внесение денежных средств в кассу ККМ).

При необходимости можно зарегистрировать выемку денежных средств из кассы ККМ (при этом автоматически формируется документ Выемка денежных средств из кассы ККМ).


Оформление чеков

Товары в табличную часть документа могут быть добавлены с использованием сканера штрихкодов, или с использованием механизма подбора товаров в документ. Для показа диалогового окна подбора необходимо нажать на кнопку Подобрать товары в документе.
При подборе товара в табличную часть документа для товаров автоматически устанавливаются отпускные (розничные) цены. В качестве розничных цен склада используются цены, зарегистрированные для того типа цен, который указан в карточке склада, с которого оформляется чек ККМ. Перед перемещением товаров в розничную точку рекомендуется создать документ Установка цен для передаваемых товаров, если это не было сделано ранее.

Для расчета с покупателями наличными или платежной картой после заполнения списка товаров в документе нажмите на соответствующую кнопку. В появившемся диалоговом окне пользователь может указать сумму полученной оплаты наличными  или зарегистрировать оплату платежной картой. Допускается оплата одновременно наличными и платежной картой по кнопке Смешанная оплата.

Если при неоплаченном чеке требуется пробить новый чек, надо нажать на кнопку Новый чек. Программа предложит отложить неоплаченный чек с резервированием до определенного времени, без резервирования либо удалить его. Отложенный чек в дальнейшем можно восстановить при помощи команды Продолжить чек.

Из формы документа по кнопке Еще - Товарный чек можно распечатать товарный чек.

Оформление возвратов

Возврат товаров розничному покупателю в течение кассовой смены оформляется документом Чек ККМ на возврат, и может быть оформлен непосредственно кассиром по команде Оформить возврат. Если кассовая смена уже закрыта, то оформить возврат товара нельзя.

Важно! В программе можно оформить только возврат товара, не являющегося лекарственным средством.

 

Корректировка в чеке ККМ


В настройках системы  Администрирование - Продажи - Розничные продажи по команде Настройки продавцов можно ввести ограничения на выполнение некоторых кассовых операций кассирам. В таких случаях данные операции можно будет выполнить старшим кассиром при добавлении ему дополнительных прав.  Например, корректировку строк в чеке.

Для случаев когда в документе при оформлении продажи по каким-либо причинам требуется удалить строки, предусмотрен механизм авторизации (требуется предвартельное включение в справочнике Настройка РМК), предоставляющей возможность по быстрой смене прав кассира без необходимости перезапуска системы.

Пример
В случае когда на кассе требуется удалить товар из чека ККМ, осуществляется считывание карты пользователя, например старшего кассира, который дополнительно авторизуется в системе.
Происходит добавление в интерфейс РМК дополнительных прав старшего кассира, которые позволяют производить удаление строк в чеке.
После выполнения операций старшим кассиром удаляются соответствующие строки в чеке ККМ и выполняется выход из режима дополнительных прав.
Кассир может продолжить свою дальнейшую работу без необходимости перезапуска системы.

 

Документы вида Чек ККМ оформляются в течение кассовой смены. По окончании кассовой смены производится закрытие кассовой смены. Документы Чек ККМ в зависимости от настроек параметра Операции при закрытии кассовой смены в разделе Администрирование-Продажи-Розничные операции или удаляются или архивируются.
При закрытии кассовой смены формируется и проводится обобщающий документ "Отчет о розничных продажах", который подводит итоги продажи в розничной торговле для конкретной ККМ (указанной в списке чеков ККМ).
