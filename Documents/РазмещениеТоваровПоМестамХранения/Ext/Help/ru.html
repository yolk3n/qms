Размещение товаров по местам хранения

Документ Размещение товаров по местам хранения служит для регистрации в системе факта размещение товаров по местам хранения внутри склада.

На основании Приказа Министерства здравоохранения и социального развития Российской Федерации от 23 августа 2010 г. N 706н "Об утверждении Правил хранения лекарственных средств" лекарственные средства размещаются на складе по пронумерованным местам хранения с использованием стеллажных карт. По данным документа Размещение товаров по местам хранения возможно вывести на печать стеллажные карточки товаров по форме №АП-73.

В шапке документа требуется указать склад, внутри которого осуществляется размещение номенклатуры по местам хранения. Размещение товаров по местам хранения осуществляется путем редактирования реквизита Место хранения(получатель) табличной части документа.
