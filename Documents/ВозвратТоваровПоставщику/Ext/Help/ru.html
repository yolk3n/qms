Возврат товаров поставщику

Документ Возврат товаров поставщику предназначен для отражения в системе факта возврата поступивших ранее позиций номенклатуры поставщику. Возврат товаров поставщику можно оформлять на основании документа поступления. При создании документа по документу поступления товаров в табличную часть на закладке Товары переносится список номенклатуры из документа-основания, а на закладке Условия возврата указывается сам документ поступления. Табличную часть документа можно заполнить данными из различных документов поступления по кнопке Заполнить - Добавить товары из поступлений, в этом случае будет открыта форма подбора документов поступлений, где есть возможность отбора документов по периоду поступления.

При проведении документа формируются движения по регистрам накопления, в которых отражается факт возврата товаров поставщику. При проведении документа контролируется наличие в свободном остатке товаров в том количестве, которое указано в документе.

Документ служит для формирования унифицированной печатных форм ТОРГ-12 (Товарная накладная) и Накладная на отпуск материалов на сторону (0504205).
