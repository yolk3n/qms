Статьи расходов

Статьи расходов указываются при регистрации всех расходов предприятия. Статьи расходов являются классификатором, в разрезе которого можно проанализировать источники возникновения расходов.
В конфигурации предусмотрены варианты распределения:

  • На себестоимость товаров
  • Не распределять - расходы по такой статье не подлежат распределению и будут списаны на указанную аналитику расходов.

Для обеспечения соответствия данных управленческого учета принципам бухгалтерского учета - вариант распределения на себестоимость товаров рекомендуется использовать только для расходов, классифицируемых в бухгалтерском учете как Транспортно-заготовительные расходы.

Расходы, распределяемые на себестоимость товаров.

Расходы по данной статье распределяются на себестоимость по указанной аналитике расходов и согласно заданному правилу распределения: пропорционально количеству или пропорционально сумме.
Если значение аналитики расходов не указано (обязательность заполнения аналитики расходов регулируется флажком Контролировать заполнение аналитики) - распределение будет идти на все товары с соответствующей аналитикой:

  • Для Склада - по всем складам (остаткам на начало месяца и поступлениям);
  • Для Номенклатуры - по всей номенклатуре;
  • Для Поступления товаров - по всем поступлениям товаров за месяц;
  • Для Перемещения товаров - по всем перемещениям товаров за месяц.

Ограничение использования статьи расходов.

Для статьи расходов может быть задан список хозяйственных операций, при отражении которых возможно указание данной статьи. При отражении других хозяйственных операций данная статья расходов будет не видна.
