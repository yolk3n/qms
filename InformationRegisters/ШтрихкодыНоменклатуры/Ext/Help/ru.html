Штрихкоды номенклатуры

Регистр сведений Штрихкоды номенклатуры используется для хранения информации о штрихкодах номенклатуры.

Наполнение регистра осуществляется по кнопке Перейти - Штрихкоды номенклатуры элемента справочника Номенклатура.

Штрихкоды для номенклатуры детализируются до уровня единиц измерения товара и серий товаров. То есть для каждого товара, имеющего определенную серию и единицу измерения можно задать отдельный штрихкод.

Новый штрихкод можно задать вручную, нажав на кнопку Создать.

Важно! Для лекарственных средств, для которых указан штрихкод упаковки товара (производителя) (в справочнике Аптечные товары), новый штрихкод создавать нужно только в том случае, если для учета лекарств в больничной аптеке используются внутреннее штрихкодирование (в том числе по сериям и единицам измерения). При работе по заполнению документов с помощью сканера штрихкода поиск в базе данных осуществляется по обоим штрихкодам.
