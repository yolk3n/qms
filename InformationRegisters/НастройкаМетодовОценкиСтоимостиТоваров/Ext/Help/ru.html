Регистр сведений "Настройка методов оценки стоимости товаров"

Метод оценки стоимости товаров устанавливается для каждой организации предприятия. Установленный метод оценки стоимости обычно действует в течение года.

Расчет себестоимости по указанному методу оценки стоимости товаров производится регламентным документом "Расчет себестоимости товаров". Регламентный документ обычно оформляется в конце каждого месяца.

В программе можно выбрать один из методов оценки стоимости товаров: "ФИФО" или "Средняя за месяц". Для каждой организации можно назначить свой метод оценки стоимости товаров.
