ABC классификация номенклатуры

В отчете выводятся результаты проведенной ABC-классификации товаров. ABC-классификации номенклатуры проводится периодически с помощью обработки Классификация номенклатуры.
Для запуска процесса выполнения классификации в обработке следует нажать кнопку Выполнить ABC-классификацию за все периоды. Классификация выполняется в соответствии с настройками периода выборки, за который будут анализироваться данные по закупкам, которые можно задать по кнопке Настройка в обработке Классификация номенклатуры. В настройках задается период классификации и количество предшествующих периодов, за которые анализируются данные.

ABC-классификация товаров проводится одновременно по следующим разделам учета:

  •  по организации в целом;
  •  по отдельным подразделениям организации, осуществляющим закупку.

ABC-классификация товаров проводится одновременно по пяти объектам классификации:

  • по действующему веществу (МНН)
  • по действующему веществу (МНН) + форма выпуска
  • по торговым наименованиям
  • по торговым наименованиям + форма выпуска
  • по номенклатуре

Соответственно, для формирования отчетов по ABC-классификации можно выбрать следующие варианты:

  •  Действующие вещества (МНН) в разрезе организаций;
  •  Действующие вещества (МНН) в разрезе подразделений организаций;
  •  Действующие вещества (МНН) с формой выпуска;
  •  Действующие вещества (МНН) с формой выпуска в разрезе организаций;
  •  Действующих вещества(МНН) с формой выпуска в разрезе подразделений организаций;
  •  Торговые наименования в разрезе организаций;
  •  Торговые наименования в разрезе подразделений организаций;
  •  Торговые наименования с формой выпуска в разрезе подразделений организаций;
  •  Торговые наименования с формой выпуска;
  •  Торговые наименования с формой выпуска в разрезе организаций;
  •  Номенклатура в разрезе организаций;
  •  Номенклатура в разрезе подразделений организаций.
