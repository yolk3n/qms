Подбор товаров в документы списания

Обработка подбора товаров в документы списания предназначена для облегчения процесса подбора номенклатуры в документы списания аптечных товаров со складов больничной аптеки.

При формировании документа по кнопке Заполнить - Подобрать товары открывается форма подбора, в которой надо выбрать товар двойным щелчком левой кнопки мыши по конкретной позиции.
Список номенклатуры для подбора товаров сформирован с информацией о наличии товаров на складе, указанном в шапке документа. Флажок Только в наличии на складе служит для отбора в список товаров, имеющихся на складе на момент заполнения документа.
Указание источника финансирования в соответствующем поле служит для отбора списка товаров по источнику финансирования.
Выбранные товары отражаются в нижнем окне формы подбора. По кнопке Перенести в документ товары попадают в табличную часть документа, после чего надо будет заполнить остальные реквизиты табличной части.

Если установить флажок Запрашивать количество по кнопке Все действия - Настройки, то при подборе товара можно сразу указать количество, единицу измерения и серию списываемого товара.

Для ускорения поиска нужного товара в форме предусмотрен отбор по определенным признакам. Для этого нужно установить флажок Фильтр по и выбрать вариант для построения фильтра. Фильтр может быть:

  • по иерархии - в этом случае можно ограничить выбор товаров в пределах группы номенклатуры, которые заданы в справочнике Номенклатура;
  • по лекарственным средствам - в этом случае можно настроить отбор товаров по группе ПКУ, действующему веществу, торговому наименованию, производителю, форме выпуска, принадлежности к спискам, источнику финансирования;
  • по виду и свойствам - в этом случае выбор товаров можно ограничить видом номенклатуры, к которому относится товар и реквизитами быстрого доступа, если они были заданы в справочник Виды номенклатуры на закладке Панель быстрого доступа.

Для быстрого поиска товара в форме подбора надо воспользоваться кнопкой Найти, в списке останутся товары, соответствующие наименованию в строке поиска. Для возврата к полному списку товаров надо воспользоваться кнопкой Отменить поиск.
