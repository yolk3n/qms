Реестр решений о вводе ЛП в гражданский оборот

Обработка позволяет получить информацию из реестра решений о вводе ЛП в гражданский оборот, обработка обеспечивает соблюдение требований Постановления Правительства РФ №60 от 28.01.2021 г. «О внесении изменения в пункт 32 Положения о системе мониторинга движения лекарственных препаратов для медицинского применения», который внес изменения в Постановление Правительства РФ №1556 «Об утверждении положения о системе мониторинга движения лекарственных препаратов для медицинского применения» http://publication.pravo.gov.ru/Document/View/0001202101290006

Для разных групп участников действуют разные правила оборота лекарств.

Для дистрибюторов

Для маркированных лекарств, выпущенных в гражданский оборот после 1 февраля 2021 года, действует упрощенный режим.
Для всех схем, включая 702, необходимо наличие положительной квитанция о фиксации сведений в ИС "Маркировка.МДЛП".
Для маркированных лекарств, выпущенные в гражданский оборот до 1 февраля 2021 года продолжает действовать уведомительный режим до момента выбытия этих лекарств из оборота.

Для аптек и медицинских организаций

Для аптек и МО уведомительный режим в части приемки товара действует, согласно Постановлению Правительства РФ №1779, бессрочно.
Аптеки и МО руководствуются абзацем вторым пункта 32, согласно которому фактом представления сведений является не наличие успешной квитанции, а факт направления сведений в систему.

Дата ввода ЛП в гражданский оборот отображается в колонке "Дата записи АИС РЗН". В случае наличия нескольких записей по одному GTIN и Номеру серии ЛП, необходимо учитывать самую раннюю дату записи АИС РЗН.

Пример 1:

В таблице реестра, для GTIN "04810201017099" и Номером серии "0000000", будет несколько строк с датами записи АИС РЗН 18.09.2020 и 04.02.2021. По скольку минимальная дата меньше 01.02.2021 г., для данного ЛП действует уведомительный режим как для дистрибюторов, так и для аптек и МО.

Пример 2:

В таблице реестра, для GTIN "04810201017099" и Номером серии "0000001", будет одна строка с датой записи АИС РЗН 04.02.2021. По скольку дата больше 01.02.2021 г., для дистрибюторов, для данного ЛП действует упрощенный режим. Для аптек и МО продолжает действовать уведомительный режим, согласно ПП РФ №1779.
