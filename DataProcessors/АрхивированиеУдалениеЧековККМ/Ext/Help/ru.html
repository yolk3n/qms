Архивирование и удаление чеков ККМ

Форма предназначена для удаления или архивирования чеков ККМ в ручном режиме. Для автоматического удаления чеков ККМ можно воспользоваться регламентным заданием Удаление чеков ККМ.

  • Для заполнения списка касс ККМ, по которым будут удаляться/архивироваться чеки ККМ необходимо указать период, за который будет происходить удаление/архивирование. В табличной части отобразится количество чеков ККМ за указанный период по каждой кассе ККМ.
  • Для удаления/архивации чеков необходимо выбрать кассы ККМ, по которым будет происходить удаление либо архивация, указать выполняемое действие Удаление чеков ККМ или Архивирование чеков ККМ и нажать кнопку Выполнить.
