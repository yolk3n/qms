Сопоставление значений с рубрикатором

В форме производится сопоставление значений текущего реквизита номенклатуры (характеристики) информационной базы со значениями реквизита рубрикатора 1С:Номенклатуры.

Возможно ручное и автоматическое сопоставление значений реквизитов номенклатуры.

Автоматическое сопоставление значений

Автоматическое сопоставление значений реквизитов информационной базы производится по команде Заполнить. При этом незаполненные значения передаются в сервис, который возвращает найденные соответствующие значения.

Ручное сопоставление значений

Для ручного сопоставления значений необходимо выбрать соответствующее значение из выпадающего списка значений в колонке Значение 1С:Номенклатура.
