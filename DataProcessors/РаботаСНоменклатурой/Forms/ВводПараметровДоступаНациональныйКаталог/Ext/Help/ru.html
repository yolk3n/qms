Параметры доступа в Национальный каталог

apikey - ключ доступа к API Национального каталога.
party_id - идентификатор компании, к которой должна быть привязана карточка товара.
Получить значения этих параметров, а так же более подробную информацию по работе с Национальным каталогом, можно по запросу в техническую поддержку на адрес support@national-catalog.ru

Регистрация gtin для товаров

Одновременно с отправкой номенклатуры в Национальный каталог можно зарегистрировать gtin для новых товаров. Данная возможность доступна если организация является членом ГС1 РУС. Выпущенный штрихкод будет закреплен за товаром в случае его успешной регистрации в Национальном каталоге.
Для того, чтобы зарегистрировать gtin для новых товаров, должен быть установлен флаг "Получать штрихкоды при регистрации товаров" в форме.
При установленном флаге "Получать штрихкоды при регистрации товаров" проверка наличия штрихкодов у выгружаемых в Национальный каталог товаров не осуществляется, для всех товаров без штрихкодов будут запрошены новые штрихкоды. В случае, если этот флаг не установлен, наличие штрихкода у товара является обязательным условием выгрузки в Национальный каталог.
