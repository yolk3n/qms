Результаты выгрузки номенклатуры

Форма предназначена для ознакомления с результатами выгрузки номенклатуры в Национальный каталог, а так же исправления ошибок выгрузки, выявленных по итогам модерации в Национальном каталоге.
Результаты выгрузки можно фильтровать по состоянию при помощи кнопок «Отклонена», «Принята», «Проверяется модератором» и «Ожидает выгрузки».
Номенклатура в состоянии «Отклонена» сопровождается описанием ошибки. Нужно принять меры по ее исправлению, затем нажать кнопку «Исправлено, выгрузить повторно», после чего номенклатура переходит в состояние «Ожидает выгрузки» и будет выгружена автоматически при следующем запуске выгрузки по расписанию, либо вручную по кнопке «Выгрузить» из формы «Помощник выгрузки».
