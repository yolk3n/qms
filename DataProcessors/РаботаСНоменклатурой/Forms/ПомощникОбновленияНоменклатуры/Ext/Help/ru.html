Помощник обновления номенклатуры

Помощник предназначен для упрощения процесса обновления номенклатуры и видов номенклатуры. Перед обновлением необходимо, чтобы Виды номенклатуры были сопоставлены с Категориями номенклатуры в сервисе, Номенклатура в базе - с Номенклатурой в сервисе.

Обновление видов номенклатуры

В списке доступны для обновления и редактирования виды номенклатуры, у которых установлен ручной режим обновления.
Для обновления необходимо перейти по гиперссылке Изменено реквизитов у каждого вида номенклатуры.
Откроется форма заполнения вида номенклатуры. Для завершения обновления необходимо нажать на кнопку Готово.

Для перехода к обновлению номенклатуры необходимо нажать кнопку Далее.

Обновление номенклатуры

В списке доступна для обновления и редактирования номенклатура, у которой установлен ручной режим обновления.
Для редактирования Обновления реквизитов необходимо перейти по гиперссылке у каждой номенклатуры.
Возможно групповое и индивидуальное обновление номенклатуры. Чтобы была обновлена только выбранная номенклатура, необходимо нажать кнопку Обновить номенклатуру.
Кнопка Обновить все предназначена для обновления всего списка номенклатуры.

Для отображения в списке Видов номенклатуры и номенклатуры с выбранным режимом автоматического обновления необходимо поставить флаг Показывать обновляемые автоматически.
