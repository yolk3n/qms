Номенклатура к выгрузке

Добавить номенклатуру к выгрузке можно из формы выбора по кнопке "Добавить", либо по отбору по кнопке "Заполнить по отбору". Рекомендуется пользоваться вторым вариантом – "Заполнить по отбору".
При добавлении номенклатуры по отбору настройки отбора сохраняются и затем используются для оповещений о появлении новых позиций к выгрузке. Чтобы отключить уведомления о новых позициях, нужно снять флаг "Оповещать о появлении новых позиций".
На закладке "Исключения" располагается список номенклатуры, которую нужно исключить из выгрузки.
