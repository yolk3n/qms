Проблемы при формировании документов

Форма отображает список учетных документов, для которых не удалось сформировать электронные документы.

В колонке "Результат" выводится гиперссылка с кратким представлением информации об ошибках. При переходе по гиперссылке открывается помощник устранения ошибок. После исправления ошибок в колонке отображается гиперссылка "Сформирован", по которой можно открыть форму представления сформированного электронного документа.

Если электронный документ формировался по команде "Отправить электронный документ", то в командной панели формы отображается кнопка "Подписать и отправить документы", которая позволяет повторно сформировать, подисать и отправить электронные документы.
