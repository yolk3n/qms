Обработка предназначена для тестирования и анализа записи из ФЭР (федеральной электронной регистратуры). Из формы обработки есть возможность отправить пакеты, аналогичные пакетам ФЭР. По окончанию тестирования в обработке есть возможность отобразить информацию об ошибках записи, если таковые были, а так же данные расписания (медицинская организация, количество специальностей, количество врачей, количество талонов и минут), которые были отправлены.
