Подбор товаров в документ списания по заказу

В нижнем окне формы отражаются товары из требования отделения. Для подбора отпускаемых товаров, соответствующих товарам из требования, можно воспользоваться кнопкой Заполнить по остаткам. В этом случае программа автоматически подберет товары к отпуску в нужном количестве. Эти товары будут отражаться в строке под соответствующими товарами из требования отделения.
Если подбор не удалось осуществить или удалось подобрать товары не в полном количестве, программа выдаст соответствующее сообщение. В этом случае можно воспользоваться списком товаров в верхнем окне формы подбора и выбрать товар, аналогичный требуемому.

Список номенклатуры для подбора товаров в верхнем окне сформирован с информацией о наличии товаров на складе, указанном в шапке документа. Флажок Только в наличии на складе служит для отбора в список товаров, имеющихся на складе на момент заполнения документа.
Указание источника финансирования в соответствующем поле служит для отбора списка товаров по источнику финансирования.

Если установить флажок Запрашивать количество по кнопке Еще - Настройки, то при подборе товара можно сразу указать количество, единицу измерения, источник финансирования и серию списываемого товара.

Для ускорения поиска нужного товара в форме предусмотрен отбор по определенным признакам. Для этого нужно установить флажок Фильтр по и выбрать вариант для построения фильтра. Фильтр может быть:

  • по иерархии - в этом случае можно ограничить выбор товаров в пределах группы номенклатуры, которые заданы в справочнике Номенклатура;
  • по лекарственным средствам - в этом случае можно настроить отбор товаров по группе ПКУ, действующему веществу, торговому наименованию, производителю, форме выпуска, принадлежности к спискам, источнику финансирования;
  • по виду и свойствам - в этом случае выбор товаров можно ограничить видом номенклатуры, к которому относится товар и реквизитами быстрого доступа, если они были заданы в справочник Виды номенклатуры на закладке Панель быстрого доступа.

Для быстрого поиска товара в форме подбора надо воспользоваться кнопкой Найти, в списке останутся товары, соответствующие наименованию в строке поиска. Для возврата к полному списку товаров надо воспользоваться кнопкой Отменить поиск.

Для побора товара, не указанного в требовании отделения, надо установить курсор на строку с надписью сверх заказа и перенести в нее нужный товар из списка в верхнем окне, либо перетащив его мышкой, либо двойным щелчком по левой кнопке мыши.
Выбранные товары отразятся в нижней части формы отбора под строками с требуемыми товарами.
По кнопке Перенести в документ они попадут в табличную часть документа отпуска в отделение.
