Оперативный мониторинг ЛС

Обработка служит для подготовки файла для экспорта данных по отчетному периоду в сегменте в формате Росздравнадзора для проекта Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н). Для экспорта данных по закупаемым препаратам из перечня ЖНВЛП необходимо:

  • загрузить номенклатуру сегмента из внешнего файла
  • сопоставить номенклатуру сегмента с номенклатурой предприятия
  • выгрузить файл специального формата с данными по отчетному периоду в сегменте по проекту Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н)

Для загрузки номенклатуры сегмента из внешнего файла необходимо сначала экспортировать действующий справочник лекарственных средств в сегменте с сайта http://mols.roszdravnadzor.ru. При загрузке данных есть возможность регулировать перечень загружаемой номенклатуры сегмента - с нулевой ценой или только с зарегистрированными ценами - путем установки признака Загружать номенклатуру с нулевой ценой. Данные из файла записываются в справочник Номенклатура поставщиков, для этого необходимо создать контрагента Росздравнадзор и указать его в качестве контрагента в обработке по проекту для мониторинга цен.

В форме сопоставления номенклатуры сегмента с номенклатурой предприятия необходимо указать номенклатуру сегмента для каждой номенклатуры предприятия, которая должна попасть в файл выгрузки для проекта по мониторингу цен.

Внимание! В файл выгрузки попадает только номенклатура, которая сопоставлена номенклатуре сегмента.

Возможны следующие режимы сопоставления номенклатуры:

  • ручной
  • автоматический

Для ручного сопоставления служит кнопка Установить связь. Для автоматического сопоставления - Автоматическое сопоставление. При нажатии на эту кнопку происходит автоматическое сопоставление номенклатуру сегмента с номенклатурой предприятия по коду Росздравнадзора PackNX. Код PackNX можно заполнить в форме элемента справочника Аптечные товары на закладке Основные данные. Если для наполнения справочника используется Регистр лекарственных средств России® в составе информационного продукта ИТС МЕДИЦИНА, то этот код загружается автоматически для товаров, имеющих зарегистрированную цену. При ручном сопоставлении перечень номенклатуры Росздравнадзора ограничен по соответствию кода PackNX номенклатуры предприятия и кодов PackNX из файла.

Для корректного заполнения файла для импорта данных в формате Росздравнадзора для проекта Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н) необходимо заполнить код поставщика VendorID в форме элемента справочника Контрагенты, который можно найти на странице сайта http://mols.roszdravnadzor.ru.

Для формирования файла с данными по отчетному периоду в сегменте в формате Росздравнадзора для проекта Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н) необходимо нажать кнопку Выгрузить. При нажатии этой кнопки вызывается вспомогательная форма для формирования содержимого файла. В этой форме необходимо указать дату (если выгружаются сведения об остатках) или период (если выгружаются сведения о закупках), организацию, от имени которой формируется выгрузка и нажать кнопку Заполнить. При этом табличная часть Данные по мониторингу цен будет автоматически заполнена учетными данными. После проверки и корректировки содержимого таблицы можно сформировать файл с помощью кнопки Выгрузить данные для мониторинга цен и импортировать его в систему мониторинга цен в ЛПУ и аптечных учреждениях на странице http://mols.roszdravnadzor.ru.
