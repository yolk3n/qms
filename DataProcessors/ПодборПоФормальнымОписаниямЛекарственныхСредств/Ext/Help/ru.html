Подбор ФОЛС

Обработка Подбор ФОЛС предназначена для облегчения процесса подбора лекарственных средств по формальным признакам (формальному описанию лекарственного средства) в требованиях отделений.

Подбор по формальному описанию лекарственного средства позволяет гибко организовать систему формирования потребностей в условиях неопределенности номенклатуры на момент требования и подбора аналогов при отпуске конкретной номенклатуры со склада или поступающей номенклатуры от поставщика.

В форме подбора в табличной части представлен перечень формальных описаний препаратов (лекарственных средств и биологически активных добавок), сформированных на основании содержимого справочника Аптечные товары. Выбранные позиции попадают в окно Подобранные товары, затем по кнопке Перенести в документ - в табличную часть документа. При установке флажка Запрашивать количество по кнопке Еще-Настройки можно будет при выборе сразу указать количество требуемого препарата.

После переноса товаров в табличную часть по двойному щелчку на позиции номенклатуры можно конкретизировать, по какому конкретно формальному признаку надо будет подбирать товары при отпуске по требованию:

  • по торговому наименованию: В этом случае при последующем подборе номенклатуры при поступлении товаров от поставщика в общем случае может быть подобрана любая подходящая номенклатура (если при этом не заполнен реквизиты Номенклатура), имеющая соответствующее торговое наименование (упаковки и производители могут быть произвольными). Если при этом указан реквизит Действующее вещество (МНН), то при подборе номенклатуры сначала будет осуществляться поиск по торговому наименованию, в случае неудачи – поиск будет продолжен по действующему веществу.
  • по действующему веществу (МНН): В этом случае степень свободы при последующем подборе номенклатуры при поступлении товаров от поставщика будет максимальной (если при этом не заполнены реквизиты "Торговое наименование" и "Номенклатура").
