Навигация по структуре классификатора

Форма подбора визуально разделена на две части: левую, где отображается иерархическая структура классификатора, и правую, содержащую детальную информацию о выделенном элементе. Навигация по структуре классификатора осуществляется путем раскрытия иерархических группировок (разделов, групп, подгрупп, т.д.). В режиме загрузки, для создания элементов в информационной базе может быть выбрано несколько отдельных позиций или целых группировок. Элементы, коды которых уже имеются в информационной базе, по умолчанию отображаются выделенными и недоступны для выбора.

Поиск элементов

Для поиска может использоваться часть наименования или описания элемента, а также код или его фрагмент, которые необходимо вводить в поле поиска.
При отображении результатов поиска представление элементов в левой части формы меняется на список. После выбора определенного элемента будет выполнен возврат к иерархическому режиму отображения классификатора с позиционированием на выбранном элементе.
Если количество найденных элементов превысит 100, результаты поиска будут выводиться постранично с указанием номера текущей страницы. Для смены страниц следует пользоваться командами В начало, Предыдущая и Следующая, которые будут автоматически добавлены на форму.

Отображение неактуальных элементов

По умолчанию отображаются только актуальные элементы. Для просмотра элементов с истекшим сроком действия, необходимо установить отметку Отображать утратившие силу позиции.
В дереве классификатора всегда отображаются только уникальные коды элементов. Если определенный код является действующим, но имеет предыдущие версии с тем же кодом и истекшими сроками действия, его неактуальные версии будут указаны в отдельной группировке Предыдущие версии в области детальной информации по элементу. Состояние настройки Отображать утратившие силу позиции не влияет на отображение группировки Предыдущие версии.
