Форма вариантов

Форма предназначена для просмотра и выбора одного из вариантов сопоставления номенклатуры информационной базы с номенклатурой контрагента.

Другие действия

Данные контрагента

В шапке формы представлена информация о номенклатуре контрагента:
- Наименование
- Характеристика
- Единица измерения
- Артикул
- Штрихкод

Варианты сопоставления

В таблице Найдено в моей базе приведен список предлагаемых вариантов сопоставления.
Если наименование, артикул или штрихкод номенклатуры совпадает с данными контрагента, то текст в соответствующем поле выделяется жирным шрифтом.

Другие действия

Если ни один из вариантов не подходит, то можно воспользоваться другими способами подбора номенклатуры:
- Выбрать из всего списка номенклатуры.
- Создать новую номенклатуру по данным контрагента.
- Подобрать / Загрузить из сервиса 1С:Номенклатура (если данный сервис используется).
