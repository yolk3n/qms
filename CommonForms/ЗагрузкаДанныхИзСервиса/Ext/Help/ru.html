Каркасные версии расширений

Каркасная версия расширения конфигурации описывает ту же структуру данных, что и полная версия, но не содержит программного кода, форм и иной интеллектуальной собственности разработчика расширения.

Каркасная версия расширения конфигурации может автоматически использоваться вместо полной версии расширения в случаях, когда применение полной версии расширения невозможно. Например:

● при обнаруженной несовместимости расширения конфигурации с новой версией конфигурации прикладного решения;

● при выгрузке приложения в файл выгрузки (для перехода в другой сервис или для загрузки в локальную версию), если у пользователя нет прав на получение файла полной версии расширения;

● при отключении полной версии расширения по инициативе его правообладателя (например, при неоплате использования платного расширения конфигурации).

Скачивание и установка каркасных версий расширений самостоятельно

Если по каким-то причинам каркасная версия не была установлена автоматически, есть возможность скачать и установить ее вручную. Для этого необходимо:

  1. В личном кабинете менеджера сервиса зайти в меню "Архивирование"
  2. Найти необходимую резервную копию, или ближайшую, если выгрузка происходила из приложения
  3. Открыть карточку резервной копии нажать кнопку "Еще" и выбрать пункт "Получить файл каркасных версий расширений"
  4. Установить эти расширения в локальную базу (как правило, это делается с помощью команды Администрирование — Печатные формы, отчеты и обработки — Расширения или Настройки — Печатные формы, отчеты и обработки — Расширения , или переходом по навигационной ссылке e1cib/app/ОбщаяФорма.Расширения)
  5. Перезайти в пустую базу для подключения расширений
  6. Запустить загрузку из архива
