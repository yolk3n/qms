«1С:Медицина. Больничная аптека»

Конфигурация "1С:Медицина. Больничная аптека" предоставляет следующие возможности:

  • Ведение учета движения номенклатуры аптечных товаров по сериям, срокам годности, документам партии (документам поступления в аптеку)
  • Учет по складам (суммовой и количественный) и по местам хранения склада (количественный)
  • Возможность ведения учета как на складах аптеки, так и в отделениях медицинской организации
  • Возможность ведения раздельного учета по источникам финансирования
  • Регистрация поступления аптечных товаров от поставщиков на склады аптеки и оприходования (ввод остатков или излишков по результатам инвентаризации)
  • Регистрация возвратов товаров поставщикам
  • Регистрация размещения аптечных товаров по местам хранения (в случае ведения количественного учета по местам хранения)
  • Регистрация отпуска в отделения, перемещений товаров между складами больничной аптеки
  • Регистрация списания аптечных товаров
  • Регистрация изготовления по лекарственным прописям
  • Регистрация фасовки готовых аптечных товаров из упаковки поставщика в упаковку больничной аптеки
  • Учет в розничном аптечном пункте медицинской организации
  • Проверка по перечню изъятых из обращения серий препаратов по письмам Росздравнадзора
  • Регистрация инвентаризации товарно-материальных ценностей  
  • Формирование заказов отделений по формальным признакам лекарственного средства (по действующему веществу или международному не патентованному наименованию, торговому наименованию, форме выпуска и т.д.)
  • Формирование развернутых аналитических отчетов по движению аптечных товаров
  • Формирование регламентированной бухгалтерской отчетности
  • Загрузка зарегистрированных цен производителей на ЖНВЛП с ИТС МЕДИЦИНА и через онлайн-сервис фирмы РЛС
  • Экспорт данных по отчетному периоду в сегменте в формате Росздравнадзора для портала "Оперативный мониторинг лекарственных средств"
  • Обмен с конфигурациями для бухгалтерского учета на платформе "1С:Предприятие": 1С:Бухгалтерия государственного учреждения 8, 1С:Бухгалтерия предприятия 8, 1С:Бухгалтерия предприятия 8 КОРП, 1С:Воинская часть 8
  • Поддержка внешнего оборудования: терминалов сбора данных, сканеров штрихкодов, электронных весов, эквайрингового оборудования и т. д.

Прикладное решение "1С:Медицина. Больничная аптека" предназначено для автоматизации деятельности больничных аптек медицинских организаций любой формы собственности (хозрасчетные и бюджетные).

Специализированное решение учитывает все особенности бизнес-процессов больничной аптеки внутри медицинской организации, которое позволяет организовать учет как в самой аптеке, так и в отделениях. Конфигурация может использоваться для любых складов медицинской организации, т.к. реализован универсальный подход для учета ТМЦ разного вида наряду со специализированными механизмы для учета лекарственных средств.

Для облегчения рутинного процесса заполнения справочника номенклатуры аптечных товаров и для актуализации информации по лекарственных средствам реализована возможность обмена с диском ИТС МЕДИЦИНА. При загрузке информации с ИТС МЕДИЦИНА в конфигурацию становится возможной ежемесячная актуализация цен на ДЛО и ЖНВЛП по лекарственным средствам, которые были переданы в конфигурацию ранее.
