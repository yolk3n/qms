Виды номенклатуры

Справочник Вид номенклатуры служит для объединения элементов справочника Номенклатура по основным признакам учета. Вид номенклатуры определяет такие характеристики элемента номенклатуры как тип (готовая лекарственная форма, товар, экстемпоральная номенклатура), группа бухгалтерского учета, разрезы учета (по сериям, партиям). В элементе справочника Вид номенклатуры задаются некоторые значения по умолчанию для реквизитов элемента номенклатуры данного вида.

Элемент справочника имеет 4 закладки.
На закладке Основное вводится наименование элемента, группа, если элементы справочника разбиты по группам, группа бухгалтерского учета, в которую будет входить номенклатура данного вида и тип номенклатуры.

Флажок Использовать серию, устанавливается, если предполагается ведения учета по сериям для номенклатуры данного вида. После установки этого флажка выбирается вариант определения серии и точность срока годности серии.
Реквизиты Политика учета серий и Политика учета серий в отделении задают правило расчета себестоимости для товара данного вида и могут принимать значения:

  • Справочное указание серий - остатки товара и себестоимость рассчитываются в целом по номенклатуре без учета серий. Указание серий в документах носит информативный характер.
  • Управление остатками серий - остатки товара в отчетах и формах подбора показываются в разрезе серий, себестоимость рассчитывается в целом по номенклатуре.
  • Учет себестоимости по сериям - остатки товара и себестоимость рассчитываются в пределах конкретной серии номенклатуры.

Флажок Использовать партии устанавливается, если для номенклатуры данного вида предполагается ведение учета по партиям. После установки этого флажка необходимо выбрать политику ведения партионного учета для аптеки и отделений:

  • Управление остатками партий - остатки товара рассчитываются в разрезе партий, Себестоимость считается без учета партий;
  • Учет себестоимости по партиям - остатки и себестоимость товара рассчитывается в пределах одной партии (документа оприходования).

На закладке Дополнительные реквизиты можно ввести дополнительные реквизиты и свойства товаров данного вида, если это предусмотрено в настройках системы.

На закладке Шаблоны наименований задаются правила формирования рабочего наименования и наименования для печати номенклатуры, относящейся к данному виду.
В шаблоне наименования для формирования наименования номенклатуры используется торговое наименование, дозировка, лекарственная форма, производитель, поэтому при создании вида номенклатуры с типом номенклатуры Товар шаблоны наименований задавать не надо.

На закладке Панель быстрого доступа можно задать реквизиты, по которым можно будет осуществить отбор товаров данного вида в форме подбора при заполнении табличных частей документов, установив фильтр по виду и свойствам.

На закладке Значения по умолчанию можно значение НДС, группу, в которую должна попасть создаваемая данного вида номенклатура, значения основной единицы учета и минимальной к отпуску из аптеки, а так же выбрать шаблоны этикетки, ценника, стеллажной карточки, если планируется их печатать.
