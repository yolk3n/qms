Склады

Справочник "Склады" предназначен для ведения списка тех складских территорий, в которых хранится товар.

Существует возможность выбора календаря, который используется для указания дней работы конкретного склада. Реквизит носит информативный характер.

Установкой флажка "Использовать места хранения" предусмотрена возможность размещения товаров по адресным местам хранения в пределах склада. Переход к списку мест хранения производится по соответствующей кнопке на панели навигации вверху формы элемента справочника "Склады".

Для каждого склада на закладке "Основное" указывается материально ответственное лицо и его должность. Данные о материально ответственном лице склада выводятся в печатных формах документов.

На закладке "Адреса, телефоны" вводится информация об адресе склада и телефоне склада.

В программе можно использовать склады следующих типов: Общий, Отделение, Розничный магазин.

В группе реквизитов Печать необходимо выбрать виды цены, в которых будут отражаться суммовые реквизиты печатных форм документов.

Для склада с типом Розничный магазин дополнительно указывается источник финансирования и вид розничной цены, по которой будет отпускаться товар.

Важно. Не рекомендуется изменять данные, если склад уже выбран в документах. Разблокировать реквизиты справочника для дальнейшего их изменения можно с помощью пункта "Разрешить редактирование реквизитов объекта" по кнопке .
