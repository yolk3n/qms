Группы доступа номенклатуры

Справочник предназначен для описания доступа к элементам справочника Номенклатура по виду доступа "Группы номенклатуры". Справочник используется в том случае, если необходимо ограничить доступ пользователям (или группе пользователей) к конкретным позициям номенклатуры.

Для менеджеров будет ограничен список доступных им позиций номенклатуры.

  • Менеджеры по закупкам лекарственных средств будут видеть только те позиции номенклатуры, для которых определена группа доступа номенклатуры, например, Лекарственные средства
  • Менеджеры по закупкам расходных материалов будут видеть только те позиции номенклатуры, для которых определена группа доступа номенклатуры Расходные материалы

    Важно. Установить ограничения по группам доступа номенклатуры возможно только в том случае, если в настройках параметров учета установлен флаг "Ограничивать доступ на уровне записей". В профиле групп доступа пользователей должен быть добавлен вид доступа "Группа доступа номенклатуры".
