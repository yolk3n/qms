Эквайринговые терминалы

Справочник Эквайринговые терминалы предназначен для хранения информации о терминалах регистрации оплат по платежным картам.

Справочник "Эквайринговые терминалы" предназначен для хранения информации о терминалах регистрации оплат по платежным картам. Терминалы разделены по организациям, для терминала указывается касса, с которой он используется.
В табличной части справочника по кнопке Добавить можно указать виды платежных карт, обслуживаемых данным терминалом.
Если эквайринговый терминал не подключен к компьютеру, то для него устанавливается флажок "Использовать без подключения оборудования".

Эквайринговые терминалы указываются в документах розничных продаж.

Примечание. Предварительно информация об используемых моделях торгового оборудования и о параметрах его подключения к рабочим местам пользователей регистрируется с помощью обработки "Подключение и настройка торгового оборудования".
