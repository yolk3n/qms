Шаблоны сводных проводок для обмена с бухгалтерией

В шаблоне проводок на закладке Настройка счетов необходимо заполнить поля КПС, Счет, КЭК для дебета и кредита проводки.
Значения КПС дебета и кредита, а так же общие реквизиты проводки ИФО и КФО можно указать как фиксированными, так и ссылками на значения из регистра Счета учета операций организации.
Значения счетов дебета и кредита можно указать так же фиксированным значением либо значениями, установленными по умолчанию.

Значения субконто выбираются из регистра накопления, являющегося источником данных в настройке хозяйственной операции, для которой создается проводка.

Количество и Источник балансовой суммы выбираются из значений показателей отражения учета соответствующей настройки хозяйственной операции и означают:

  •  Количество – количество товара из соответствующего регистра накопления ;
  •  Сумма и Сумма без НДС – явно указанные в документах суммы (например, при возврате товара, отпуске на сторону);
  •  Стоимость и Стоимость без НДС – рассчитанная себестоимость товара с НДС и без НДС соответственно;
  •  Стоимость (регл.) – себестоимость товара с НДС или без НДС в зависимости от того, принят НДС к вычету или нет при поступлении товара.

Для отражения отрицательной суммы проводки надо установить флажок Изменять знак операции при отражении в учете.

Для включения проводки в конкретную настройку отражения в бухгалтерском учете необходимо по ссылке Использовать в обменах выбрать соответствующую настройку.
При обмене проводками в БГУ будут формироваться только те проводки, шаблоны которых включены в настройку отражения.

На закладке Дополнительный отбор можно задать дополнительные условия формирования проводки по заданному шаблону
