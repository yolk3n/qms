Настройки хозяйственных операций

Справочник хозяйственных операций содержит предопределенный перечень хозяйственных операций, для которых есть возможность настроить проводки для обмена с бухгалтерией.

В справочнике определяется тип данных, по которым формируется приход и расход при отражении операции в учете.

На закладке Отражение в учете в реквизите Источник данных указан регистр накопления, из которого будут браться данные для заполнения реквизитов проводки.

На закладке Связанные документы отражаются документы, формирующие данную хозяйственную операцию.

На закладке Показатели для отражения в учете флажком отмечены количественные и суммовые показатели, которые доступны для выбора источников количества и балансовой суммы при настройке шаблона проводки.

  •  Количество – количество товара из соответствующего регистра накопления;
  •  Сумма и Сумма без НДС – явно указанные в документах суммы (например, при возврате товара, отпуске на сторону);
  •  Стоимость и Стоимость без НДС – рассчитанная себестоимость товара с НДС и без НДС соответственно;
  •  Стоимость (регл.) – себестоимость товара с НДС или без НДС в зависимости от того, принят НДС к вычету или нет при поступлении товара.
