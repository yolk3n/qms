Регистр лекарственных средств

Справочник Аптечные товары (Регистр лекарственных средств) предназначен для хранения зарегистрированных в Российской Федерации аптечных товаров в части лекарственных средств и биологически активных добавок, используемых в больничной аптеке.

Справочник Аптечные товары используется в конфигурации совместно со справочником Номенклатура, служит для упорядочения и унификации классификационных признаков лекарственных средств и биологически активных добавок. Использование справочника Аптечные товары в конфигурации позволяет расширить реквизитный состав номенклатуры лекарственных средств и биологически активных добавок и оперировать формальным описанием лекарственного средства (биологически активной добавки), что позволяет получать расширенную аналитическую отчетность по движению аптечных товаров и эффективно организовать работу с синонимами (аналогами) лекарственных средств.

Синонимы – это лекарственные средства, имеющие одинаковое действующее вещество (международное непатентованное наименовании), но различные торговые наименования и виды упаковок.

Для автоматического наполнения справочника списком аптечных товаров на основании утвержденных в РФ реестров предусмотрена возможность загрузки из Регистра лекарственных средств России® в составе информационного продукта ИТС МЕДИЦИНА или сайта ИТС http://its.1c.ru/.

Подробнее об информационном продукте ИТС МЕДИЦИНА и его использовании можно ознакомиться на сайте фирмы 1С и в разделе О справочнике РЛС на диске.

Загрузка лекарственных средств используется для:

  • получения новых элементов;
  • актуализации информации о ранее загруженных элементах.

По кнопке ИТС: Медицина - Настроить можно выбрать источник загрузки элементов справочника: с диска ИТС или сайта ИТС.

Загрузка данных происходит в 2 этапа:
- по кнопке ИТС: Медицина - Открыть поиск товаров в базе РЛС открывается справочник лекарственных средств, где необходимо выбрать интересующие товары;
- по кнопке ИТС: Медицина - Принять данные из базы РЛС - отобранные товары загружаются в базу, о чем свидетельствует информация на экране.

При загрузке с диска открывается окно приложения ИТС МЕДИЦИНА (при условии, что диск вставлен в дисковод, и произведена его установка), и информационная база 1С:Предприятие переходит в состояние ожидания получения данных с диска. После поиска нужного лекарственного средства по кнопке Передать происходит его передача в базу данных. Во время загрузки данных программа выдает сообщение о ходе загрузки. После получения всех отобранных товаров можно закрыть обработку приема данных и вернуться в базу РЛС для поиска и отбора новой порции данных.

При загрузке с диска можно загрузить пакет элементов, сформированный по определенному критерию при работе с "ИТС МЕДИЦИНА".
Формирование пакета для выгрузки сведений об аптечных товарах c диска ИТС МЕДИЦИНА подробно описано в разделе О справочнике РЛС - Пакетная выгрузка данных на диске ИТС в разделе Медицина - Регистр лекарственных средств России.



Для загрузки лекарственных средств с сайта ИТС надо открыть поиск товаров в базе РЛС.
После отбора на сайте необходимых лекарственных средств и помещения их в корзину надо принять данные в базу.
Для загрузки следующей «порции» данных необходимо закрыть обработку и снова открыть поиск товаров в базе с помощью кнопки ИТС:Медицина - Открыть поиск товаров в базе ИТС.

Элементы справочника, созданные путем загрузки информации из справочника РЛС доступны для редактирования при установке признака Ручная корректировка. Для того, чтобы установить этот признак необходимо нажать ссылку Включить режим ручной корректировки в форме элемента справочника. После установки признака ручной корректировки в форме списка появляется специальная пиктограмма ручной корректировки.

Для актуализации информации о ранее загруженных элементах следует выделить список элементов с помощью нажатия и удерживания клавиши Ctrl и одновременных щелчков левой кнопкой мыши по элементам в форме списка справочника (или одновременного нажатия комбинации клавиш Ctrl+A для выделения всего списка сразу), после чего нажать кнопку ИТС:МЕДИЦИНА - Обновить товары из базы РЛС на верхней панели справочника. При обновлении элементов справочника с установленным признаком Ручная корректировка содержимое элемента справочника не обновляется, но обновляются данные по регистрационному удостоверению и по принадлежности к спискам ЖНВЛП, ДЛО, наркотических, психотропных и ядов. Для обновления содержимого элемента с признаком ручной корректировки надо нажать кнопку ИТС:МЕДИЦИНА - Обновить товары из базы РЛС с отменой ручных изменений.

При создании новых элементов справочника в ручном режиме необходимо заполнить реквизиты элемента справочника на закладке Основные данные.

Заполнение классификатора аптечных товаров в части лекарственных средств в ручном режиме рекомендуется использовать только в случае отсутствия аптечного товара в Государственном реестре лекарственных средств. Такая ситуация возможна, например, для лекарственных средств, изготавливаемых в больничной аптеке, или для субстанций, которые централизовано расфасовываются из крупной тары в более мелкую, для последующей поставки отделениям больницы.

Код – код элемента в справочнике «Классификатор аптечных товаров». В типовой конфигурации проверка на уникальность и автоматическая нумерация кода заданы в пределах всего справочника.
Наименование полное – формируется на основании реквизитов "Действующее вещество или МНН", "Торговое наименование", "Лекарственная форма", данным по видам упаковки. После заполнения этих данных наименование элемента сформируется автоматически при нажатии на кнопку справа от реквизита Наименование.

На закладке Основные данные указываются общие сведения об элементе классификатора аптечных товаров.

  • Действующие вещества или МНН – название лекарственного средства, принятое Всемирной организацией здравоохранения (ВОЗ), выбирается из справочника Действующие вещества или МНН. Этот справочник не рекомендуется заполнять вручную даже при ручном вводе элемента классификатора аптечных товаров, рекомендуется загрузить с диска ИТС МЕДИЦИНА любой элемент с идентичным действующим веществом (МНН) и воспользоваться автоматически созданным элементом справочника Действующие вещества или МНН. В дальнейшем это позволит легко получать отчеты по классификационным признакам лекарственных средств (биологически активных добавок) и работать с механизмом синонимов (аналогов).
  • Торговое наименование – выбирается из справочника Торговые наименования. Этот справочник не рекомендуется заполнять вручную даже при ручном вводе элемента классификатора аптечных товаров, рекомендуется загрузить с диска «ИТС МЕДИЦИНА» любой элемент с идентичным торговым наименованием и воспользоваться автоматически созданным элементом справочника Торговые наименования.
  • Фирма-производитель – наименование фирмы-производителя аптечного товара, выбирается из справочника Производители.
  • Страна производителя – наименование страны производителя, выбирается из справочника Классификатор стран мира.
  • Фирма-регистратор – наименование фирмы-регистратора аптечного товара, выбирается из справочника Производители.
  • Страна регистратора – наименование страны регистратора, выбирается из справочника Классификатор стран мира.
  • Срок годности – срок годности аптечного товара.
  • Штрихкод - штрихкод упаковки товара (производителя), зарегистрированного в государственном реестре.
  • Код PackNX - код Росздравнадзора для экспорта данных по проекту Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н). Если для наполнения справочника используется Регистр лекарственных средств России® в составе информационного продукта ИТС МЕДИЦИНА, то этот код загружается автоматически для товаров, имеющих зарегистрированную цену.
  • Код DrugID - уникальный код Росздравнадзора для экспорта данных по проекту Мониторинг цен в ЛПУ и аптечных учреждениях (пр.277н). Если для наполнения справочника используется Регистр лекарственных средств России® в составе информационного продукта ИТС МЕДИЦИНА, то этот код загружается автоматически для товаров, имеющих зарегистрированную цену. Подробнее об экспорте данных по отчетному периоду в сегменте в формате Росздравнадзора в описании обработки Проект по мониторингу цен.
  • Реквизит Форма выпуска содержит такие справочные сведения о форме выпуска (лекарственной форме) препарата, количестве доз в упаковке, массе лекарственной формы, концентрации и т.д. Ознакомиться с ними можно раскрыв это поле по кнопке . Лекарственная форма выбирается из справочника Лекарственные формы. Этот справочник не рекомендуется заполнять вручную даже при ручном вводе элемента классификатора аптечных товаров, рекомендуется загрузить с диска ИТС МЕДИЦИНА любой элемент с идентичной лекарственной формой и воспользоваться автоматически созданным элементом справочника Лекарственные формы.
  • Реквизит Упаковка содержит сведения о первичной, вторичной и третичной упаковке аптечного товара. Увидеть их можно, раскрыв это поле по кнопке . Единицы измерения упаковок определяют перечень единиц измерения аптечного товара. Единица измерения содержания товара в первичной упаковке является базовой единицей аптечного товара. Она же будет являться базовой единицей элемента справочника Номенклатура, созданного на основании этого аптечного товара.

На основной закладке так же пиктограммами обозначена принадлежность лекарственного средства к спискам (сильнодействующих и ядовитых веществ, ДЛО, ЖНВЛП, безрецептурного отпуска, А и Б).

Под знаком принадлежности к списку ЖНВЛП по ссылке Цены можно увидеть зарегистрированные цены на лекарственное средство.

Закладка "Классификация"

На этой закладке указываются классификационные признаки препарата по различным общероссийским или международным классификаторам.

  • Группа ПКУ - указывается группа предметно-количественного учета, автоматически заполняется при загрузке элемента из базы РЛС на основании сведений о принадлежности препарата к спискам "Наркотическое вещество", "Сильнодействующие и яды" или на основании действующего вещества (по спиртам). Предопределенная группа "Прочие ЛС" устанавливается автоматически для лекарств, не принадлежащих ни к одному из списков.
  • Товары фарм.рынка - классификация аптечных товаров с точки зрения принадлежности в государственным реестрам (лекарственные средства, биологически активные добавки, товары фармацевтического рынка).перечень фармакологических групп, к которым принадлежит лекарственное средство. Табличная часть заполняется из справочника «Фармакологические группы».
  • АТХ - анатомо-терапевтическо-химическая классификация по международной системе классификации лекарственных средств.
  • Фармдействие - фармацевтическое действие препарата.
  • Коды ОКДП, ОКПД, ОКПД2 - список кодов по Общероссийскому классификатору видов экономической деятельности, продукции.

На форме элемента справочника расположены дополнительные элементы управления. При нажатии на кнопку ИТС:МЕДИЦИНА можно открыть окно информации об аптечном товаре в базе РЛС или найти синонимы этого препарата.

Через панель навигации формы элемента справочника можно перейти к просмотру и редактированию таких дополнительных сведений о лекарственном средстве, как статус Состояния регистрационного удостоверения (дата аннулирования регистрации, дата и документ перерегистрации), По кнопке Дополнительно можно увидеть базовую единицу препарата, номер элемента в справочнике РЛС и ссылку на описание препарата в базе РЛС.

На панели инструментов списка справочника по кнопке ИТС:МЕДИЦИНА открывается меню действий с использованием диска ИТС и сайта ИТС. Команды этого меню предусматривают поиск и загрузку новых элементов в справочник Аптечные товары, обновление загруженных ранее элементов, поиск синонимов.

С помощью команд Открыть номенклатуру и Создать номенклатуру в группе меню Номенклатура в форме списка справочника можно создать новый элемент справочника Номенклатура на основании аптечного товара или открыть существующий элемент номенклатуры, связанный с данным аптечным товаром.
