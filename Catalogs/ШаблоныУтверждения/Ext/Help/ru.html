Шаблоны утверждения

Шаблоны бизнес-процесса Утверждение располагаются в разделе Органайзер - Шаблоны процессов. В списке видов бизнес-процессов следует выбрать вид Утверждение. Шаблоны данного вида будут показаны в списке, расположенном справа от списка видов бизнес-процессов. В списке с помощью команд "Создать" и "Изменить" контекстного меню можно создать новый или отредактировать имеющийся шаблон.

В шаблоне бизнес-процесса Утверждение можно указать различные данные, которые будут использованы при создании нового бизнес-процесса на основании документа.

  • Наименование шаблона;
  • Наименование - наименование бизнес-процесса, создаваемого по данному шаблону;
  • Описание;
  • Кому - выбирается из справочника пользователей;
  • Срок;
  • Важность.

Для автоматического применения этого шаблона нужно привязать его к виду документов. В этом случае при создании нового бизнес-процесса на основании документа по виду документа будет автоматически найден и использован нужный шаблон. Для этого в панели навигации с помощью команды "Назначен видам документов" откройте список видов документов и дополните его записями с нужными видами.

Поля карточки бизнес-процесса, заполненные из шаблона, становятся недоступными для ручного изменения пользователем.

Поля "Наименование шаблона", "Ответственный", "Комментарий" носят информационно-справочный характер и не влияют на создание новых бизнес-процессов по этому шаблону.

Любой пользователь, имеющий права на работу с бизнес-процессами, может создать персональный шаблон бизнес-процесса. В карточке персонального шаблона установлен флаг "Персональный шаблон". Персональный шаблон доступен для использования и изменения только его автором. Общие шаблоны бизнес-процессов создаются ответственным за НСИ. Общие шаблоны доступны для использования всем пользователям, но для изменения или создания только ответственному за НСИ. Персональные шаблоны выделяются в списке шаблонов цветом текста.

В поле "Кому" может быть указан исполнитель, которому адресуется задача.
Исполнителей бизнес-процесса можно назначить персонально, либо с помощью ролевой адресации по кнопке Подобрать - Роли иди Добавить - Роли исполнителей.
