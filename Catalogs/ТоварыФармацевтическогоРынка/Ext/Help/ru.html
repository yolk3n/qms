Товары фармацевтического рынка

Справочник предназначен для хранения списка групп по классификации товаров фармацевтического рынка в "Регистре лекарственных средств в России®" фирмы РЛС. Принадлежность к группе по классификации товаров фармацевтического рынка задается для элемента справочника "Аптечные товары" на закладке "Основные данные".

Справочник загружается с диска ИТС:МЕДИЦИНА по кнопке Загрузить классификатор с диска ИТС

Реквизиты справочника

Реквизиты справочника
  • Наименование - указывается наименование группы товаров фармацевтического рынка.
  • Группа>- указывается группа в справочнике, в которую входит групп товаров фармацевтического рынка.
  • Флажок "Лекарственные средства" - устанавливается для групп товаров фармацевтического рынка, которые являются лекарственными средствами.
  • >Флажок "Перевязка"> - устанавливается для групп товаров фармацевтического рынка, которые являются перевязочными средствами.
