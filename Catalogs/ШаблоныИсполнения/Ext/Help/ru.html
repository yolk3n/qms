Шаблоны исполнения

Шаблоны бизнес-процесса Исполнение располагаются в разделе Органайзер - Шаблоны процессов. В списке видов бизнес-процессов следует выбрать вид Исполнение. Шаблоны данного вида будут показаны в списке, расположенном справа от списка видов бизнес-процессов. В списке с помощью команд Создать и Изменить контекстного меню можно создать новый или отредактировать имеющийся шаблон.

  • Наименование шаблона;
  • Описание поручения (общее);
  • Наименование - наименование бизнес-процесса, создаваемого по данному шаблону;
  • Срок;
  • Важность;
  • Список исполнителей, для каждого из которых можно установить отдельный срок и описание (если требуется);
  • Ответственный исполнитель - выбирается из справочника пользователей;
  • Проверяющий - выбирается из справочника пользователей;
  • Контролер - выбирается из справочника пользователей.

Для автоматического применения этого шаблона нужно привязать его к виду документов. В этом случае при создании нового бизнес-процесса на основании документа по виду документа будет автоматически найден и использован нужный шаблон. Для этого в панели навигации с помощью команды "Назначен видам документов" откройте список видов документов и дополните его записями с нужными видами. Для того, чтобы использовать условную привязку шаблона бизнес-процесса к виду документов, необходимо указать, на основании какого условия маршрутизации будет осуществляться эта привязка. Если условие не указано, привязка будет осуществляться во всех случаях.

Поля карточки бизнес-процесса, заполненные из шаблона, становятся недоступными для ручного изменения пользователем.

Поля "Наименование шаблона", "Ответственный" и "Комментарий" носят информационно-справочный характер и не влияют на создание новых бизнес-процессов по этому шаблону.

Любой пользователь, имеющий права на работу с бизнес-процессами, может создать персональный шаблон бизнес-процесса. В карточке персонального шаблона установлен флаг "Персональный шаблон". Персональный шаблон доступен для использования и изменения только его автором. Общие шаблоны бизнес-процессов создаются ответственным за НСИ. Общие шаблоны доступны для использования всем пользователям, но для изменения или создания только ответственному за НСИ. Персональные шаблоны выделяются в списке шаблонов цветом текста.

В списке исполнителей (внизу формы) можно указать одного или несколько исполнителей бизнес-процессов, созданных по этому шаблону. Если исполнителей несколько, то одного из них с помощью команды "Ответственный исполнитель" можно назначить ответственным. В этом случае этот исполнитель может контролировать выполнение работы соисполнителей. Для этого в карточке задачи, показываемой ответственному исполнителю выводится список состояний выполнения работ соисполнителей.
Исполнителей бизнес-процесса можно назначить персонально, либо с помощью ролевой адресации по кнопке Подобрать - Роли иди Добавить - Роли исполнителей.

В таблице исполнителей в графе Условие можно задать условие маршрутизации, при выполнении которого к исполнителю будет направляться задача бизнес-процесса. Если условие не задано, то задачи бизнес-процессов, созданных по этому условию будут всегда направляться данному исполнителю.
