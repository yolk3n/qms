Единицы измерения

Справочник служит для хранения единиц измерения аптечных товаров.

В поле «Наименование» задается произвольное наименование единицы измерения. Для каждой единицы измерения задается тип.
Типы единиц измерения:

Единицы измерения товара:

Единицы измерения характеристик лекарственной формы:

Единицы измерения товара:
  • Экономические единицы (Единица измерения штучного товара);
  • Единица массы;
  • Единица объема;
  • Единицы длины;
  • Единицы времени;
  • Единицы площади;
  • Технические единицы.

Единицы измерения характеристик лекарственной формы:
  • Единица измерения концентрации;
  • Единица измерения действия.

Типы «Единица измерения концентрации», «Единица измерения действия» используются для заполнения единиц измерения справочника «Аптечные товары»: единица измерения концентрации, единиц действия, размера лекарственной формы.

Единицы измерения товара доступны для задания типа базовой единицы измерения элементов справочника "Номенклатура" .

После первого запуска программы происходит начальное заполнение справочника.
Если в нем нет нужной единицы, ее можно добавить из Классификатора единиц измерения с помощью кнопки Подобрать из классификатора или ввести вручную.

Единицы измерения, загруженные при первом запуске программы нельзя изменять и дублировать.
