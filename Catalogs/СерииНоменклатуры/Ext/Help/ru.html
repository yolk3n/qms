Серии номенклатуры

В конфигурации предусмотрено ведение учета по сериям номенклатуры.
Справочник «Серии номенклатуры» подчинен справочнику «Номенклатура».
Необходимость ведения учета по сериям для позиции номенклатуры определяется в справочнике «Виды номенклатуры» политикой ведения учета по сериям.

Для каждой серии необходимо указать следующие параметры:

  • Серийный номер
  • Срок годности
