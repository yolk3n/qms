Забраковки серий

Справочник предназначен для хранения списка серий, изъятых из обращения лекарственных средств в соответствии с решениями Росздравнадзора.

Справочник может заполняться как по данным РЛС (вручную; с помощью данных, поставляемых с диска ИТС:Медицина или с сайта ИТС),
так и по данным сервиса МДЛП по кнопке Загрузить забракованные серии в соответствующем подменю.

Для решений о приостановке / отмене приостановке загруженных по данным РЛС доступны следующие команды в подменю РЛС -> Обработать выделенные строки:

  • Поиск забракованных серий для выделенных элементов - произвести сопоставление выделенных забракованных серий и серий номенклатуры с помощью обработки "Поиск забракованных серий".
  • Объединить выделенные забракованные серии - перенести все сопоставленные серии номенклатуры в одну из выделенных забракованных серий. Рекомендуется использовать для объединения введенных ранее вручную забракованных серий с загружаемой с диска ИТС. При объединении серии номенклатуры, ранее прикрепленные к указанным забракованным сериям (источникам), прикрепляются к забракованной серии-приемнику, и удаляются старые связи.
  • Пометить как обработанные/необработанные - можно отметить строки, по которым уже был произведен поиск. Строки, отмеченные как обработанные отражаются в списке тонким шрифтом.

Для быстрого поиска нужной серии в справочнике предусмотрена строка отбора по номеру и/или дате приказа.
Флажок Только необработанные служит для отражения в списке забракованных серий по данным РЛС, по которым еще не производился поиск в базе данных.

Для решений о приостановке / отмене приостановки, загруженных по данным МДЛП, сопоставление забракованных серий выполняется автоматически при загрузке.
