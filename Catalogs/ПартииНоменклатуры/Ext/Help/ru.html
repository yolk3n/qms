Партии номенклатуры

Справочник предназначен для хранения списка партий номенклатуры, поступающей в организацию.

Каждый элемент справочника создается автоматически на основании документа поступления (документа, образующего партию) при условии, что по данной номенклатурной позиции ведется учет по документам оприходования ( определяется политикой использования партий в справочнике "Виды номенклатуры", к которой относится данная номенклатурная позиция).

В регистре учета движений аптечных товаров на складах (регистр «Товары на складах») одно из измерений указывает партию номенклатуры, определяемое элементом справочника "Партии номенклатуры", если по данной номенклатурной позиции ведется учет по документам оприходования. Таким образом, по движениям аптечных товаров возможно ведение аналитического учета в разрезе партий.
