Места хранения

Справочник "Места хранения" используется для разделения складов на области хранения в соответствии с различными условиями хранения.

Важно. Места хранения используются только для тех складов, на которых предусмотрено адресное хранение товаров. Для таких складов должна быть установлена опция "Использовать места хранения".

Данные места хранения лекарственного средства указываются при печати стеллажной карточки.
