Договоры контрагентов

Справочник предназначен для хранения договоров взаиморасчетов, заключенных с контрагентами. Данный справочник подчинен справочнику «Контрагенты». Все хозяйственные операции с контрагентами оформляются с обязательным указанием договора.

Справочник может включать неограниченное число уровней иерархии, что позволяет группировать договоры контрагентов по определенным категориям и осуществлять их быстрый поиск.

С помощью механизма Присоединенные файлы к элементу справочника можно присоединить отсканированную копию бумажного документа.
