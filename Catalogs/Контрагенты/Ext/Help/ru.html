Контрагенты

Справочник "Контрагенты" используется для хранения списка организаций-поставщиков аптечных товаров. Справочник используется при ведении аналитического учета по закупкам аптечных товаров.

Справочник может включать неограниченное число уровней иерархии, что позволяет группировать контрагентов по определенным категориям и осуществлять их быстрый поиск.

При вводе нового контрагента Возможно заполнение основных реквизитов по ИНН на основании данных ЕГРЮЛ/ЕГРИП.
Рекомендуем выработать правила записи рабочего наименования контрагентов, которые позволят устранить повторный ввод контрагента в справочник и обеспечат быстрый поиск его в списке контрагентов.
