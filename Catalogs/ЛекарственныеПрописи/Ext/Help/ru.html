Лекарственные прописи

Справочник содержит перечень лекарственных прописей, по которым производится изготовление лекарственных средств в больничной аптеке.

Реквизиты шапки:

Реквизиты шапки:
  • Продукция – указывается номенклатура лекарственного средства, изготавливаемого в больничной аптеке, выбирается из справочника «Номенклатура»,
  • Количество – указывается количество изготавливаемой номенклатуры, в расчете на которое указаны количественные нормы ингредиентов и других товаров в прописи,
  • Единица измерения - указывается единица измерения изготавливаемой продукции, для которой указано количество,
  • Наименование – указывается наименование лекарственной прописи,
  • Статус - указывается статус прописи (Действует, В разработке или Отменена). В документе Изготовление по лекарственной прописи можно выбрать только пропись со статусом действует.
  • Основная пропись - флажок для отметки основной прописи в случае, когда для одной и той же продукции может использоваться несколько прописей.

В табличной части по кнопке Заполнить - Подобрать товары по формальным описаниям указывается перечень ингредиентов лекарственных средств, входящих в лекарственную пропись. Ингредиенты (субстанции) указываются в терминах формального описания лекарственного средства, указываются признаки лекарственных средств: действующее вещество (МНН) и/или торговое наименование, а также форма выпуска. В дальнейшем при проведении изготовления лекарственного средства программа будет автоматически подбирать нужные торговые позиции по складским остаткам с соответствии с формальным описанием лекарственного средства.
Для включения в пропись перечня вспомогательных товаров, используемых при изготовлении по лекарственной прописи, например, тары, в которую разливаются лекарственные средства, надо воспользоваться кнопкой Заполнить - Подобрать товары.

В табличной части необходимо указать количество ингредиента, единицу измерения и норму естественной убыли ингредиента при изготовлении продукции (этот реквизит носит информативный характер).

Следует заполнить реквизит Ответственный – 'это сотрудник, ответственный за утверждение лекарственной прописи. По умолчанию ответственным устанавливается пользователь программы.
